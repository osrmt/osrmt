Reports

RMT has a very good reporting module.

It is possible to define SQL report and XML reports.

To generate SQL report, please define the following

-       Report Name

-       Report Directory

-       Parameters for the report

-       Output file name

-       The SQL statements

To generate XML report, please define the following

-       Select Script

-       Sort Script

-       Field script

-       Image script

-       The SQL statements

-       The Query

There are also system predefined reports that are available. The output of these reports are available in pdf or HTML format.
